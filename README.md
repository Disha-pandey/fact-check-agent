# fact-check-agent

AI-powered fact-checking web application built using Streamlit, OpenAI, Tavily, and Serper APIs.

## Features
- Upload PDF documents
- Extract factual claims
- Verify claims using AI and web search
- Generate verification reports
- Download CSV report

## Tech Stack
- Python
- Streamlit
- OpenAI API
- Tavily API
- Serper API
